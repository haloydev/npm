# Haloy

Simple deployment tool for Docker-based applications.

## Installation

```bash
npm install -g haloy
```

Or use with npx:

```bash
npx haloy version
```

## Usage

```bash
haloy [command] [flags]
```

For full documentation, visit [haloy.dev](https://haloy.dev).

## Supported Platforms

- macOS (ARM64, x64)
- Linux (ARM64, x64)
- Windows (ARM64, x64)
